专业版特性

Pigsty 专业版包含以下功能特性

  • 系统定制能力：针对精确的主流Linux操作系统发行版大小版本提供预制的离线安装包
  • 离线安装能力：在没有互联网访问的环境中（断网环境）实现 Pigsty 的完整安装
  • 内核定制能力：允许用户切换 PostgreSQL 生命周期内大版本的内核（12 - 15）
  • 内核定制能力：允许用户使用其他 PG 系内核，替换原生 PG。
    • Babelfish：基于 PG Microsoft SQL Server 线缆协议级兼容能力
    • IvorySQL：基于 PG 提供 Oracle 语法/类型/存储过程兼容能力
    • PolarDB：国产信创内核支持（类Oracle RAC）
    • 后续将进一步提供完整的 Greenplum，Neon 兼容能力
  • PG扩展插件：333 个 PG Extension，提供 PG 12-17 完整的扩展插件支持
  • 完整功能模块：提供所有功能模块：
    • Docker
    • Kafka
    • MinIO
    • DuckDB
    • PostgresML
    • Supabase
    • Kubernetes
    • VictoriaMetrics & VictoriaLogs
  • 架构咨询一次，根据环境、资源与需求，提供数据库架构建议与配置文件参考
  • 国产操作系统支持：提供国产信创操作系统支持选项
  • 芯片架构支持：提供 ARM64 架构支持选项

  • Pigsty CLI 命令行工具（TBD）
