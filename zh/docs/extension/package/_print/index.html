1 - 扩展构建环境准备

如何准备构建 Pigsty 扩展所需的环境：EL8/9，Debian12，Ubuntu22 虚拟机

准备虚拟机

想要在 EL / Debian 环境下构建 PGML RPM 包，需要准备虚拟机环境，Pigsty 提供了 build-ext.yml 模板，可用于准备构建所需的虚拟机环境

cd pigsty
make build
./node.yml -i files/pigsty/build-ext.yml -t node_repo,node_pkg

Pigsty 默认提供 EL8，EL9，Debian12，Ubuntu22 四台虚拟机，并安装构建所需的依赖软件包：

环境与代理

准备环境变量，最好配置一个 HTTP 代理，例如，假设你的本地环境中，有一台可用的代理服务器： http://192.168.0.104:8118 （需要换成你自己的地址）。

那么 EL 环境下的环境变量配置如下：

PROXY=http://192.168.0.104:8118
