扩展生态

关于 Pigsty 中提供的 PostgreSQL 插件，功能介绍，详细说明，注意事项，以及构建指南。

Pigsty 提供了丰富的 PostgreSQL 扩展插件支持，总共有 284 个可用扩展插件（以系统视图计）；如果以扩展软件包衡量，则有 190 个额外的扩展 RPM 包与 174 个 DEB 在 Pigsty 中可用。

Pigsty 在 PostgreSQL 社区官方的 PGDG YUM / APT 仓库基础上，维护了一个额外的 YUM / APT 扩展仓库 ，收录补充了大量 PG 生态的实用扩展插件，并尽可能对齐了 PGDG YUM/APT 提供的扩展功能集合。

RPM扩展

Pigsty 在 EL系操作系统 上共有 230 个扩展，其中包括 73 个 PostgreSQL 自带扩展 和 157 个额外的 RPM 扩展，其中由 Pigsty 维护的占 34 个。

统计以 EL8 版本为基准，有 6 个扩展尚未针对 PG 16 完成适配（带有 ❋ 标记） —— 故目前实际可用扩展为 224 个。
