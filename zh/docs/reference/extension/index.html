扩展列表

本文列出了 Pigsty 支持的 PostgreSQL 扩展插件，以及这些插件在不同系统下的支持情况。
Module:
  • PGSQL
Categories:
  • Extension

Pigsty 提供了丰富的 PostgreSQL 扩展插件支持，包括 230 个 RPM扩展 与 188 个 DEB扩展。 Pigsty 总共有 254 个可用扩展，其中含 PostgreSQL 自带的 73 个内置扩展），与由 Pigsty维护的 37 个 RPM 扩展。

RPM扩展

Pigsty 在 EL系操作系统 上共有 230 个扩展，其中包括 73 个 PostgreSQL 自带扩展 和 157 个额外的 RPM 扩展，其中由 Pigsty 维护的占 37 个。

统计以 EL8 版本为基准，有 7 个扩展尚未针对 PG 16 完成适配（带有 ❋ 标记） —— 故目前实际可用扩展为 223 个。
