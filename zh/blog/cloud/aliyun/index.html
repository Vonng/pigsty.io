ж·ҳе®қеҲ·дёҚеҮәиҒҠеӨ©еӣҫзүҮпјҢй—ӘйҖҒдёҠдј дёҚдәҶжҺҘеҚ•еҮӯжҚ®пјҢе……з”өжЎ©з”ЁдёҚдәҶпјҢеҺҹзҘһеҸ‘дёҚеҮәйӘҢиҜҒз ҒпјҢйҘҝдәҶд№ҲдёӢдёҚдәҶеҚ•пјҢйӘ‘жүӢиҝӣдёҚдәҶзі»з»ҹпјҢзӮ№дёҚдәҶеӨ–еҚ–гҖҒеҒңиҪҰеңәдёҚжҠ¬жқҶгҖҒи¶…еёӮж— жі•з»“иҙҰгҖӮз”ҡиҮіжңүзҡ„еӯҰж Ўеӣ жӯӨж— жі•з”ЁжҷәиғҪе…¬е…ұжҙ—иЎЈжңәе’ҢејҖж°ҙжңәгҖӮж— ж•°еңЁе‘Ёжң«дј‘жҒҜдёӯзҡ„з ”еҸ‘дёҺиҝҗз»ҙдәәе‘ҳиў«е–Ҡиө·жқҘеҠ зҸӯжҺ’йҡңвҖҰвҖҰ

еҢ…жӢ¬йҮ‘иһҚдә‘гҖҒж”ҝеҠЎдә‘еңЁеҶ…зҡ„еҢәеҹҹд№ҹжІЎжңүе№ёе…ҚгҖӮйҳҝйҮҢдә‘еә”иҜҘж„ҹеҲ°дёҮе№ёпјҡж•…йҡңдёҚжҳҜеҸ‘з”ҹеңЁеҸҢеҚҒдёҖеҪ“еӨ©пјҢд№ҹдёҚеңЁиЎҷй—ЁдёҺй’ұеә„зҡ„е·ҘдҪңж—¶й—ҙж®өпјҢеҗҰеҲҷеӨ§е®¶иҜҙдёҚе®ҡиғҪдёҠз”өи§ҶзңӢж•…йҡңеӨҚзӣҳдәҶгҖӮ

еҺҹеӣ жҳҜд»Җд№Ҳпјҹ

е°Ҫз®ЎйҳҝйҮҢдә‘иҮід»Ҡд»ҚжңӘз»ҷеҮәдёҖд»ҪдәӢеҗҺж•…йҡңеӨҚзӣҳжҠҘе‘ҠпјҢдҪҶиҖҒеҸёжңәж №жҚ®зҲҶзӮёеҚҠеҫ„пјҢе°ұи¶ід»ҘеҲӨж–ӯеҮәй—®йўҳеңЁе“ӘйҮҢдәҶ вҖ”вҖ” Auth пјҲи®ӨиҜҒ / йүҙжқғ / RAMпјүгҖӮ

еӯҳеӮЁи®Ўз®—зЎ¬д»¶ж•…йҡңгҖҒжңәжҲҝж–ӯз”өиҝҷдәӣй—®йўҳжңҖеӨҡеҪұе“ҚеҚ•дёӘеҸҜз”ЁеҢәпјҲAZпјүпјҢзҪ‘з»ңж•…йҡңжңҖеӨҡеҪұе“ҚдёҖдёӘеҢәеҹҹпјҲRegionпјүпјҢиғҪи®©е…ЁзҗғжүҖжңүеҢәеҹҹеҗҢж—¶еҮәй—®йўҳзҡ„иӮҜе®ҡдёҚдјҡжҳҜиҝҷдәӣй—®йўҳпјҢиҖҢжҳҜи·ЁеҢәеҹҹе…ұз”Ёзҡ„дә‘еҹәзЎҖи®ҫж–Ҫз»„д»¶гҖӮ вҖ”вҖ” жһҒеӨ§жҰӮзҺҮжҳҜ AuthпјҢдҪҺжҰӮзҺҮжҳҜе…¶д»–иҜёеҰӮи®ЎйҮҸд»ҳиҙ№д№Ӣзұ»зҡ„е…ЁеұҖжҖ§жңҚеҠЎгҖӮ

йҳҝйҮҢдә‘зҡ„дәӢд»¶иҝӣеұ•е…¬е‘ҠпјҡеҮәй—®йўҳзҡ„жҳҜжҹҗдёӘеә•еұӮжңҚеҠЎз»„д»¶пјҢиҖҢдёҚжҳҜзҪ‘з»ңгҖҒжңәжҲҝзЎ¬д»¶зҡ„й—®йўҳгҖӮ

ж №еӣ еҮәеңЁ Auth дёҠзҡ„еҸҜиғҪжҖ§жңҖеӨ§пјҢжңҖзӣҙжҺҘзҡ„иҜҒжҳҺе°ұжҳҜпјҡж·ұеәҰдёҺ Auth йӣҶжҲҗзҡ„дә‘жңҚеҠЎпјҡеҜ№иұЎеӯҳеӮЁ OSS пјҲзұ» S3пјүпјҢиЎЁж јеӯҳеӮЁ OTS пјҲзұ» DynamoDBпјүпјҢд»ҘеҸҠе…¶д»–ж·ұеәҰдҫқиө– Auth зҡ„жңҚеҠЎжң¬иә«еҸҜз”ЁжҖ§зӣҙжҺҘеҸ—еҲ°еҪұе“ҚгҖӮиҖҢжң¬иә«иҝҗиЎҢдёҚдҫқиө– Auth зҡ„дә‘иө„жәҗпјҢжҜ”еҰӮдә‘жңҚеҠЎеҷЁ ECS / дә‘ж•°жҚ®еә“ RDS д»ҘеҸҠзҪ‘з»ңд»Қз„¶еҸҜд»ҘвҖңжӯЈеёёвҖқиҝҗиЎҢпјҢз”ЁжҲ·еҸӘжҳҜж— жі•йҖҡиҝҮжҺ§еҲ¶еҸ°дёҺAPIеҜ№е…¶иҝӣиЎҢз®ЎзҗҶдёҺеҸҳжӣҙгҖӮжӯӨеӨ–пјҢдёҖдёӘеҸҜд»ҘжҺ’йҷӨзҡ„и®ЎйҮҸд»ҳиҙ№жңҚеҠЎеҮәй—®йўҳзҡ„зҺ°иұЎжҳҜпјҡж•…йҡңжңҹй—ҙиҝҳжңүз”ЁжҲ·жҲҗеҠҹд»ҳж¬ҫи–…дёӢдәҶECSзҡ„зҫҠжҜӣгҖӮ

е°Ҫз®ЎдёҠйқўзҡ„еҲҶжһҗиҝҮзЁӢеҸӘжҳҜдёҖз§ҚжҺЁж–ӯпјҢдҪҶе®ғдёҺжөҒдј еҮәжқҘзҡ„еҶ…йғЁж¶ҲжҒҜзӣёеҗ»еҗҲпјҡи®ӨиҜҒжҢӮдәҶеҜјиҮҙжүҖжңүжңҚеҠЎејӮеёёгҖӮиҮідәҺи®ӨиҜҒжңҚеҠЎжң¬иә«еҲ°еә•жҳҜжҖҺд№ҲжҢӮзҡ„пјҢеңЁе°ёжЈҖжҠҘе‘ҠеҮәжқҘеүҚжҲ‘д»¬д№ҹеҸӘиғҪзҢңжөӢпјҡдәәдёәй…ҚзҪ®еӨұиҜҜзҡ„еҸҜиғҪжҖ§жңҖеӨ§ вҖ”вҖ” еӣ дёәж•…йҡңдёҚеңЁеёёи§„еҸҳжӣҙеҸ‘еёғзӘ—еҸЈдёӯпјҢд№ҹжІЎжңүзҒ°еәҰз”ҹж•ҲпјҢдёҚеӨӘеғҸжҳҜд»Јз Ғ/дәҢиҝӣеҲ¶еҸ‘еёғгҖӮдҪҶе…·дҪ“жҳҜе“ӘйҮҢй…ҚзҪ®еӨұиҜҜпјҡиҜҒд№ҰпјҢй»‘зҷҪеҗҚеҚ•пјҢеҫӘзҺҜдҫқиө–жӯ»й”ҒпјҢиҝҳжҳҜд»Җд№Ҳе…¶д»–дёңиҘҝпјҢе°ұдёҚзҹҘйҒ“дәҶгҖӮ

е…ідәҺж №еӣ зҡ„е°ҸйҒ“ж¶ҲжҒҜд№ҹжј«еӨ©йЈһиҲһпјҢжҜ”еҰӮпјҢжңүдәәиҜҙ вҖқжқғйҷҗзі»з»ҹжҺЁдәҶдёӘй»‘еҗҚеҚ•и§„еҲҷпјҢй»‘еҗҚеҚ•и§„еҲҷз»ҙжҠӨеңЁOSSдёҠпјҢи®ҝй—®OSSеҸүйңҖиҰҒи®ҝй—®жқғйҷҗзі»з»ҹпјҢз„¶еҗҺжқғйҷҗзі»з»ҹеҸҲиҰҒи®ҝй—®OSSпјҢе°ұе°¬дҪҸдәҶвҖңгҖӮвҖқгҖӮиҝҳжңүдәәиҜҙ вҖңиҝҷж¬ЎеҸҢ11жңҹй—ҙпјҢжҠҖжңҜдәәе‘ҳиҝһз»ӯдёҖе‘ЁеҠ зҸӯеҠ зӮ№пјҢеҸҢ11дёҖиҝҮпјҢеӨ§е®¶йғҪжқҫжҮҲдәҶгҖӮжңүдёӘж–°жүӢеҶҷдәҶдёҖж®өд»Јз ҒпјҢжӣҙж–°дәҶз»„д»¶пјҢеҜјиҮҙдәҶжң¬ж¬Ўзҡ„ж–ӯзҪ‘вҖңпјҢиҝҳжңүдәәиҜҙпјҡвҖқйҳҝйҮҢдә‘жүҖжңүжңҚеҠЎз”Ёзҡ„йғҪжҳҜеҗҢдёҖдёӘйҖҡй…Қз¬ҰиҜҒд№ҰпјҢиҜҒд№ҰжҚўй”ҷдәҶвҖңгҖӮ

еҰӮжһңжҳҜеӣ дёәиҝҷдәӣеҺҹеӣ еҜјиҮҙ Auth жҢӮжҺүпјҢйӮЈеҸҜзңҹжҳҜиҚүеҸ°зҸӯеӯҗеҲ°е®¶дәҶгҖӮе°Ҫз®Ўеҗ¬дёҠеҺ»еҫҲзҰ»и°ұпјҢдҪҶиҝҷж ·зҡ„е…ҲдҫӢе№¶дёҚе°‘гҖӮеҶҚж¬Ўејәи°ғпјҢиҝҷдәӣи·Ҝиҫ№зӨҫж¶ҲжҒҜд»…дҫӣеҸӮиҖғпјҢе…·дҪ“дәӢж•…еҺҹеӣ иҜ·д»ҘйҳҝйҮҢдә‘е®ҳж–№з»ҷеҮәзҡ„еӨҚзӣҳеҲҶжһҗжҠҘе‘ҠдёәеҮҶгҖӮ

еҪұе“ҚжҳҜд»Җд№Ҳпјҹ

и®ӨиҜҒ/йүҙжқғжҳҜжңҚеҠЎзҡ„еҹәзҹіпјҢеғҸиҝҷж ·зҡ„еҹәзЎҖжҖ§з»„д»¶дёҖж—ҰеҮәзҺ°й—®йўҳпјҢеҪұе“ҚжҳҜе…ЁеұҖжҖ§гҖҒзҒҫйҡҫжҖ§зҡ„гҖӮиҝҷдјҡеҜјиҮҙж•ҙдёӘдә‘з®ЎжҺ§йқўдёҚеҸҜз”ЁпјҢдјӨе®ідјҡзӣҙжҺҘеҶІеҮ»жҺ§еҲ¶еҸ°пјҢAPIпјҢд»ҘеҸҠж·ұеәҰдҫқиө– Auth еҹәзЎҖи®ҫж–Ҫзҡ„жңҚеҠЎ вҖ”вҖ” жҜ”еҰӮе…¬жңүдә‘дёҠзҡ„еҸҰдёҖдёӘеҹәзҹіжҖ§жңҚеҠЎпјҢеҜ№иұЎеӯҳеӮЁ OSSгҖӮ

д»ҺйҳҝйҮҢдә‘е…¬е‘ҠдёҠжқҘзңӢпјҢеҘҪеғҸеҸӘжңү вҖңеҮ дёӘжңҚеҠЎпјҲ OSSгҖҒOTSгҖҒSLSгҖҒMNSпјүеҸ—еҲ°еҪұе“ҚпјҢеӨ§йғЁеҲҶдә§е“ҒеҰӮECSгҖҒRDSгҖҒзҪ‘з»ңзӯүзҡ„е®һйҷ…иҝҗиЎҢдёҚеҸ—еҪұе“Қ вҖқгҖӮдҪҶеҜ№иұЎеӯҳеӮЁ OSS иҝҷж ·зҡ„еҹәзҹіжҖ§жңҚеҠЎеҮәдәҶй—®йўҳпјҢеёҰжқҘзҡ„зҲҶзӮёеҚҠеҫ„жҳҜйҡҫд»ҘжғіиұЎзҡ„пјҢз»қйқһвҖңдёӘеҲ«жңҚеҠЎеҸ—еҲ°еҪұе“ҚвҖқ е°ұиғҪж•·иЎҚиҝҮеҺ» вҖ”вҖ” иҝҷе°ұеғҸжұҪиҪҰзҡ„жІ№з®ұйғҪзқҖзҒ«дәҶпјҢиҜҙеҸ‘еҠЁжңәе’ҢиҪ®еӯҗд»Қз„¶еңЁиҪ¬жҳҜжІЎжңүж„Ҹд№үзҡ„гҖӮ

еҜ№иұЎеӯҳеӮЁ OSS иҝҷж ·зҡ„жңҚеҠЎжҳҜйҖҡиҝҮдә‘еҺӮе•ҶеҢ…иЈ…зҡ„ HTTP API еҜ№еӨ–жҸҗдҫӣжңҚеҠЎзҡ„пјҢеӣ жӯӨеҝ…з„¶ж·ұеәҰдҫқиө–и®ӨиҜҒз»„д»¶пјҡдҪ йңҖиҰҒAK/SK/IAM зӯҫеҗҚжүҚиғҪдҪҝз”Ёиҝҷдәӣ HTTP APIпјҢиҖҢ Auth ж•…йҡңе°ҶеҜјиҮҙиҝҷзұ»жңҚеҠЎжң¬иә«дёҚеҸҜз”ЁгҖӮ

еҜ№иұЎеӯҳеӮЁ OSS е®һеңЁжҳҜеӨӘйҮҚиҰҒдәҶпјҢеҸҜд»ҘиҜҙжҳҜдә‘и®Ўз®—зҡ„вҖңе®ҡд№үжҖ§жңҚеҠЎвҖқпјҢд№ҹи®ёжҳҜе”ҜдёҖиғҪеңЁжүҖжңүдә‘дёҠеҹәжң¬иҫҫжҲҗе…ұиҜҶж ҮеҮҶзҡ„жңҚеҠЎгҖӮдә‘еҺӮе•Ҷзҡ„еҗ„з§ҚвҖңдёҠеұӮвҖқжңҚеҠЎжҲ–еӨҡжҲ–е°‘йғҪзӣҙжҺҘ/й—ҙжҺҘең°дҫқиө– OSSпјҢдҫӢеҰӮ ECS/ RDS иҷҪз„¶еҸҜд»ҘиҝҗиЎҢпјҢдҪҶ ECS еҝ«з…§е’Ң RDS еӨҮд»Ҫжҳҫз„¶жҳҜж·ұеәҰдҫқиө– OSS зҡ„пјҢCDN еӣһжәҗжҳҜдҫқиө– OSS зҡ„пјҢеҗ„дёӘжңҚеҠЎзҡ„ж—Ҙеҝ—еҫҖеҫҖд№ҹжҳҜеҶҷе…Ҙ OSS зҡ„гҖӮ

д»ҺзҺ°иұЎдёҠзңӢпјҢж ёеҝғеҠҹиғҪи·ҹ OSS ж·ұеәҰз»‘е®ҡзҡ„йҳҝйҮҢдә‘зӣҳе°ұжҢӮзҡ„еҫҲжғЁзғҲпјҢж ёеҝғеҠҹиғҪи·ҹ OSS е…ізі»дёҚеӨ§зҡ„жңҚеҠЎпјҢжҜ”еҰӮй«ҳеҫ·ең°еӣҫе°ұжІЎеҗ¬иҜҙжңүд»Җд№ҲеӨ§еҪұе“ҚгҖӮеӨ§йғЁеҲҶзӣёе…іеә”з”Ёзҡ„зҠ¶жҖҒжҳҜпјҢдё»дҪ“еҸҜд»ҘжӯЈеёёжү“ејҖиҝҗиЎҢпјҢдҪҶжҳҜе’ҢеӣҫзүҮеұ•зӨәпјҢж–Үд»¶дёҠдј /дёӢиҪҪж–Үд»¶иҝҷзұ»жңүе…ізҡ„еҠҹиғҪе°ұдёҚеҸҜз”ЁдәҶгҖӮ

жңүдёҖдәӣе®һи·өеҮҸиҪ»дәҶеҜ№ OSS зҡ„еҶІеҮ»пјҡжҜ”еҰӮйҖҡеёёиў«и®ӨдёәжҳҜдёҚе®үе…Ёзҡ„ вҖ”вҖ” дёҚиө°и®ӨиҜҒзҡ„ Public еӯҳеӮЁжЎ¶е°ұдёҚеҸ—еҪұе“ҚпјӣCDN зҡ„дҪҝз”Ёд№ҹзј“йҮҠдәҶ OSS зҡ„й—®йўҳпјҡж·ҳе®қе•Ҷе“ҒеӣҫзүҮиө° CDN зј“еӯҳиҝҳеҸҜд»ҘжӯЈеёёзңӢеҲ°пјҢдҪҶжҳҜд№°е®¶иҒҠеӨ©и®°еҪ•йҮҢе®һж—¶еҸ‘йҖҒзҡ„еӣҫзүҮзӣҙжҺҘиө° OSS е°ұжҢӮдәҶгҖӮ

дёҚд»…д»…жҳҜ OSSпјҢе…¶д»–ж·ұеәҰйӣҶжҲҗдҫқиө– Auth зҡ„дә‘жңҚеҠЎд№ҹдјҡжңүиҝҷж ·зҡ„й—®йўҳпјҢжҜ”еҰӮ OTSпјҢSLSпјҢMNS зӯүзӯүгҖӮдҫӢеҰӮеҜ№ж Ү DynamoDB зҡ„иЎЁж јеӯҳеӮЁжңҚеҠЎ OTS еҗҢж ·еҮәзҺ°дәҶй—®йўҳгҖӮиҝҷйҮҢжңүдёҖдёӘйқһеёёйІңжҳҺзҡ„еҜ№жҜ”пјҢеғҸ RDS for PostgreSQL / MySQL иҝҷж ·зҡ„дә‘ж•°жҚ®еә“жңҚеҠЎдҪҝз”Ёзҡ„жҳҜж•°жҚ®еә“иҮӘиә«зҡ„и®ӨиҜҒжңәеҲ¶пјҢжүҖд»ҘдёҚеҸ—дә‘еҺӮе•Ҷ Auth жңҚеҠЎж•…йҡңеҪұе“ҚгҖӮз„¶иҖҢ OTS жІЎжңүиҮӘе·ұзҡ„жқғйҷҗзі»з»ҹпјҢиҖҢжҳҜзӣҙжҺҘдҪҝз”Ё IAM/RAMпјҢдёҺдә‘еҺӮе•Ҷзҡ„ Auth ж·ұеәҰз»‘е®ҡпјҢеӣ жӯӨеҸ—еҲ°дәҶеҶІеҮ»гҖӮ

жҠҖжңҜдёҠзҡ„еҪұе“ҚжҳҜдёҖж–№йқўпјҢжӣҙйҮҚиҰҒзҡ„жҳҜдёҡеҠЎеҪұе“ҚгҖӮж №жҚ®йҳҝйҮҢдә‘зҡ„ жңҚеҠЎзӯүзә§еҚҸи®®пјҲSLAпјүпјҢ3дёӘеҚҠе°Ҹж—¶зҡ„ж•…йҡңдҪҝеҫ—еҪ“жңҲеҗ„жңҚеҠЎеҸҜз”ЁжҖ§жҢҮж ҮйҷҚиҮі 99.5%гҖӮиҗҪе…Ҙз»қеӨ§еӨҡж•°жңҚеҠЎиө”еҒҝж ҮеҮҶзҡ„дёӯй—ҙжЎЈдҪҚдёҠпјҢд№ҹе°ұжҳҜиө”еҒҝз”ЁжҲ·жңҲеәҰжңҚеҠЎиҙ№з”Ё 25% ~ 30% зҡ„д»ЈйҮ‘еҲёгҖӮзү№ж®Ҡзҡ„жҳҜиҝҷдёҖж¬Ўж•…йҡңзҡ„еҢәеҹҹиҢғеӣҙе’ҢжңҚеҠЎиҢғеӣҙжҳҜе…ЁйғЁпјҒ

йҳҝйҮҢдә‘ OSS SLA

еҪ“з„¶йҳҝйҮҢдә‘д№ҹеҸҜд»Ҙдё»еј иҜҙиҷҪз„¶ OSS / OTS иҝҷдәӣжңҚеҠЎжҢӮдәҶпјҢдҪҶд»–д»¬зҡ„ ECS/RDS еҸӘжҢӮдәҶз®ЎжҺ§йқўпјҢдёҚеҪұе“ҚжӯЈеңЁиҝҗиЎҢзҡ„жңҚеҠЎ,жүҖд»ҘдёҚеҪұе“Қ SLAгҖӮдёҚиҝҮиҝҷз§ҚиЎҘеҒҝеҚідҫҝзңҹзҡ„е…ЁйғЁиҗҪең°д№ҹжІЎеҮ дёӘй’ұпјҢжӣҙеғҸжҳҜдёҖз§Қе®үжҠҡжҖ§зҡ„е§ҝжҖҒпјҡжҜ•з«ҹе’Ңз”ЁжҲ·зҡ„дёҡеҠЎжҚҹеӨұжҜ”пјҢиө”дёӘжңҚеҠЎжңҲж¶Ҳ 25%д»ЈйҮ‘еҲёз®Җзӣҙе°ұжҳҜдёҖз§ҚзҫһиҫұгҖӮ

жҜ”иө·з”ЁжҲ·дҝЎд»»гҖҒжҠҖжңҜеЈ°жңӣд»ҘеҸҠе•ҶиӘүжҠҳжҚҹиҖҢиЁҖпјҢиө”зҡ„йӮЈзӮ№д»ЈйҮ‘еҲёзңҹзҡ„з®—дёҚдёҠдёүз“ңдёӨжһЈгҖӮиҝҷж¬ЎдәӢд»¶еҰӮжһңеӨ„зҗҶдёҚеҪ“пјҢдјҡжҲҗдёәе…¬жңүдә‘жӢҗзӮ№зә§еҲ«зҡ„ж Үеҝ—жҖ§дәӢд»¶гҖӮ

иҜ„и®әдёҺи§ӮзӮ№пјҹ

й©¬ж–Ҝе…Ӣзҡ„жҺЁзү№ X е’Ң DHH зҡ„ 37 Signal йҖҡиҝҮдёӢдә‘зңҒдёӢдәҶеҚғдёҮзҫҺе…ғзңҹйҮ‘зҷҪй“¶пјҢеҲӣйҖ дәҶйҷҚжң¬еўһж•Ҳзҡ„вҖңеҘҮиҝ№вҖқпјҢи®©дёӢдә‘ејҖе§ӢжҲҗдёәдёҖз§ҚжҪ®жөҒгҖӮдә‘дёҠзҡ„з”ЁжҲ·еңЁеҜ№зқҖиҙҰеҚ•зҠ№иұ«зқҖжҳҜеҗҰиҰҒдёӢдә‘пјҢжңӘдёҠдә‘зҡ„з”ЁжҲ·жӣҙжҳҜеҶ…еҝғзә з»“гҖӮеңЁиҝҷж ·зҡ„иғҢжҷҜдёӢпјҢдҪңдёәжң¬еңҹдә‘йўҶеҜјиҖ…зҡ„йҳҝйҮҢдә‘еҸ‘з”ҹеҰӮжӯӨйҮҚеӨ§ж•…йҡңпјҢеҜ№дәҺзҠ№иұ«и§ӮжңӣиҖ…зҡ„дҝЎеҝғж— з–‘жҳҜжІүйҮҚзҡ„жү“еҮ»гҖӮжҒҗжҖ•жӯӨж¬Ўж•…йҡңдјҡжҲҗдёәе…¬жңүдә‘жӢҗзӮ№зә§еҲ«зҡ„ж Үеҝ—жҖ§дәӢд»¶гҖӮ

йҳҝйҮҢдә‘дёҖеҗ‘д»Ҙе®үе…ЁзЁіе®ҡй«ҳеҸҜз”ЁиҮӘеұ…пјҢдёҠе‘ЁиҝҳеҲҡеңЁдә‘ж –еӨ§дјҡдёҠеҗ№жһҒиҮҙзЁіе®ҡжҖ§д№Ӣзұ»зҡ„зүӣйҖјгҖӮдҪҶжҳҜж— ж•°жүҖи°“зҡ„зҡ„зҒҫеӨҮпјҢй«ҳеҸҜз”ЁпјҢеӨҡжҙ»пјҢеӨҡдёӯеҝғпјҢйҷҚзә§ж–№жЎҲпјҢиў«дёҖж¬ЎжҖ§е…ЁйғЁеҮ»з©ҝпјҢжү“з ҙдәҶNдёӘ9зҘһиҜқгҖӮеҰӮжӯӨеӨ§иҢғеӣҙгҖҒй•ҝж—¶й—ҙгҖҒеҪұе“ҚйқўеҰӮжӯӨе№ҝзҡ„ж•…йҡңпјҢжӣҙжҳҜеҲӣдёӢдәҶдә‘и®Ўз®—иЎҢдёҡзҡ„еҺҶеҸІи®°еҪ•гҖӮ

иҝҷж¬Ўж•…йҡңжҸӯзӨәеҮәе…ій”®еҹәзЎҖи®ҫж–Ҫзҡ„е·ЁеӨ§йЈҺйҷ©пјҢеӨ§йҮҸдҫқжүҳдәҺе…¬жңүдә‘зҡ„зҪ‘з»ңжңҚеҠЎзјәд№ҸжңҖеҹәжң¬зҡ„иҮӘдё»еҸҜжҺ§иғҪеҠӣпјҡеҪ“ж•…йҡңеҸ‘з”ҹж—¶жІЎжңүд»»дҪ•иҮӘж•‘иғҪеҠӣпјҢйҷӨдәҶзӯүжӯ»еҒҡдёҚдәҶеҲ«зҡ„дәӢжғ…гҖӮз”ҡиҮіеҢ…жӢ¬йҮ‘иһҚдә‘е’Ңж”ҝеҠЎдә‘еңЁеҶ…д№ҹеҗҢж ·еҮәзҺ°дәҶжңҚеҠЎдёҚеҸҜз”ЁгҖӮеҗҢж—¶пјҢе®ғд№ҹеҸҚжҳ еҮәдәҶеһ„ж–ӯдёӯеҝғеҢ–еҹәзЎҖи®ҫж–Ҫзҡ„и„ҶејұжҖ§пјҡдә’иҒ”зҪ‘иҝҷдёӘеҺ»дёӯеҝғеҢ–зҡ„дё–з•ҢеҘҮи§ӮзҺ°еңЁдё»иҰҒжҳҜеңЁе°‘ж•°еҮ дёӘеӨ§е…¬еҸё/дә‘еҺӮе•ҶжӢҘжңүзҡ„жңҚеҠЎеҷЁдёҠиҝҗиЎҢ вҖ”вҖ” жҹҗдёӘдә‘еҺӮе•Ҷжң¬иә«жҲҗдёәдәҶжңҖеӨ§зҡ„дёҡеҠЎеҚ•зӮ№пјҢиҝҷеҸҜдёҚжҳҜдә’иҒ”зҪ‘и®ҫи®Ўзҡ„еҲқиЎ·пјҒ

жӣҙдёәдёҘеі»зҡ„жҢ‘жҲҳжҒҗжҖ•иҝҳеңЁеҗҺйқўпјҢе…Ёзҗғз”ЁжҲ·иҝҪзҙўиө”й’ұдәӢиҝҳе°ҸпјҢзңҹжӯЈиҰҒе‘Ҫзҡ„жҳҜеңЁеҗ„дёӘеӣҪе®¶йғҪеңЁејәи°ғж•°жҚ®дё»жқғзҡ„ж—¶еҖҷпјҢеҰӮжһңеӣ дёәеңЁдёӯеӣҪеўғеҶ…зҡ„жҹҗдёӘжҺ§еҲ¶дёӯеҝғй…ҚзҪ®еӨұеҪ“еҜјиҮҙе…Ёзҗғж•…йҡңзҡ„иҜқпјҢпјҲеҚіпјҡдҪ зңҹзҡ„еҚЎдәҶеҲ«дәәзҡ„и„–еӯҗпјүеҫҲеӨҡжө·еӨ–е®ўжҲ·дјҡз«ӢеҚійҮҮеҸ–иЎҢеҠЁиҝҒз§»еҲ°еҲ«зҡ„дә‘дҫӣеә”е•ҶдёҠпјҡиҝҷе…ід№ҺеҗҲи§„жҖ§пјҢдёҺеҸҜз”ЁжҖ§ж— е…ігҖӮ

ж №жҚ®жө·жҒ©жі•еҲҷпјҢдёҖж¬ЎдёҘйҮҚж•…йҡңзҡ„иғҢеҗҺжңүеҮ еҚҒж¬ЎиҪ»еҫ®дәӢж•…пјҢеҮ зҷҫиө·жңӘйҒӮе…Ҳе…ҶпјҢд»ҘеҸҠдёҠеҚғжқЎдәӢж•…йҡҗжӮЈгҖӮеҺ»е№ҙеҚҒдәҢжңҲйҳҝйҮҢдә‘йҰҷжёҜжңәжҲҝзҡ„еӨ§ж•…йҡңе·Із»ҸжҡҙйңІеҮәжқҘи®ёеӨҡй—®йўҳпјҢз„¶иҖҢдёҖе№ҙеҗҺеҸҲз»ҷдәҶз”ЁжҲ·дёҖдёӘжӣҙеӨ§зҡ„жғҠе–ңпјҲеҗ“пјҒпјүгҖӮиҝҷж ·зҡ„дәӢж•…еҜ№дәҺйҳҝйҮҢдә‘зҡ„е“ҒзүҢеҪўиұЎз»қеҜ№жҳҜиҮҙе‘Ҫжү“еҮ»пјҢз”ҡиҮіеҜ№ж•ҙдёӘиЎҢдёҡзҡ„еЈ°иӘүйғҪжңүдёҘйҮҚзҡ„жҚҹе®ігҖӮйҳҝйҮҢдә‘еә”иҜҘе°Ҫеҝ«з»ҷз”ЁжҲ·дёҖдёӘи§ЈйҮҠдёҺдәӨд»ЈпјҢеҸ‘еёғиҜҰз»Ҷзҡ„ж•…йҡңеӨҚзӣҳжҠҘе‘ҠпјҢи®Іжё…жҘҡеҗҺз»ӯж”№иҝӣжҺӘж–ҪпјҢжҢҪеӣһз”ЁжҲ·зҡ„дҝЎд»»гҖӮ

жҜ•з«ҹиҝҷз§Қи§„жЁЎзҡ„ж•…йҡңпјҢдёҚжҳҜвҖңжүҫдёӘдёҙж—¶е·ҘиғҢй”…пјҢжқҖдёӘзЁӢеәҸе‘ҳзҘӯеӨ©вҖқ иғҪи§ЈеҶізҡ„дәӢпјҢCEO еҫ—дәІиҮӘеҮәйқўйҒ“жӯүи§ЈеҶігҖӮCloudflare жңҲеҲқзҡ„з®ЎжҺ§йқўж•…йҡңеҗҺпјҢCEO з«ӢеҚіж’°еҶҷдәҶиҜҰз»Ҷзҡ„дәӢеҗҺеӨҚзӣҳеҲҶжһҗпјҢжҢҪеӣһдәҶдёҖдәӣеЈ°иӘүгҖӮеҸҜдёҚе№ёзҡ„жҳҜпјҢйҳҝйҮҢдә‘з»ҸиҝҮдәҶеҮ иҪ®иЈҒе‘ҳпјҢдёҖе№ҙиҝһжҚўдёүиҪ® CEO пјҢжҒҗжҖ•е·Із»ҸйҡҫжңүиғҪеҮәжқҘжүӣдәӢиғҢиҙЈзҡ„дәәдәҶгҖӮ

иғҪеӯҰеҲ°д»Җд№Ҳпјҹ
