еүҚиЁҖ

ж•°жҚ®е®Ңж•ҙжҖ§жҳҜжһҒе…¶йҮҚиҰҒзҡ„пјҢдҪҶз”ұеә”з”ЁдҝқиҜҒзҡ„ж•°жҚ®е®Ңж•ҙжҖ§е№¶дёҚжҖ»жҳҜйӮЈд№Ҳйқ и°ұпјҡдәәдјҡзҠҜеӮ»пјҢзЁӢеәҸдјҡеҮәй”ҷгҖӮеҰӮжһңиғҪйҖҡиҝҮж•°жҚ®еә“зәҰжқҹжқҘејәеҲ¶ж•°жҚ®е®Ңж•ҙжҖ§йӮЈжҳҜеҶҚеҘҪдёҚиҝҮдәҶпјҡеҗҺз«ҜзЁӢеәҸе‘ҳдёҚз”ЁеҶҚжӢ…еҝғз«һжҖҒжқЎд»¶еҜјиҮҙзҡ„еҫ®еҰҷй”ҷиҜҜпјҢж•°жҚ®еҲҶжһҗеёҲд№ҹеҸҜд»ҘеҜ№ж•°жҚ®иҙЁйҮҸе……ж»ЎдҝЎеҝғпјҢдёҚйңҖиҰҒйӘҢиҜҒдёҺжё…жҙ—гҖӮ

е…ізі»еһӢж•°жҚ®еә“йҖҡеёёдјҡжҸҗдҫӣPRIMARY KEY, FOREIGN KEY, UNIQUE, CHECKзәҰжқҹпјҢз„¶иҖҢе№¶дёҚжҳҜжүҖжңүзҡ„дёҡеҠЎзәҰжқҹйғҪеҸҜд»Ҙз”ЁиҝҷеҮ з§ҚзәҰжқҹиЎЁиҫҫгҖӮдёҖдәӣзәҰжқҹдјҡзЁҚеҫ®еӨҚжқӮдёҖдәӣпјҢдҫӢеҰӮзЎ®дҝқIPзҪ‘ж®өиЎЁдёӯзҡ„IPиҢғеӣҙдёҚеҸ‘з”ҹйҮҚеҸ пјҢзЎ®дҝқеҗҢдёҖдёӘдјҡи®®е®ӨдёҚдјҡеҮәзҺ°йў„е®ҡж—¶й—ҙйҮҚеҸ пјҢзЎ®дҝқең°зҗҶеҢәеҲ’иЎЁдёӯеҗ„дёӘеҹҺеёӮзҡ„иҫ№з•ҢдёҚдјҡйҮҚеҸ гҖӮдј з»ҹдёҠиҰҒе®һзҺ°иҝҷз§ҚдҝқиҜҒжҳҜзӣёеҪ“еӣ°йҡҫзҡ„пјҡиӯ¬еҰӮUNIQUEзәҰжқҹе°ұж— жі•иЎЁиҫҫиҝҷз§ҚиҜӯд№үпјҢCHECKдёҺеӯҳеӮЁиҝҮзЁӢжҲ–иҖ…и§ҰеҸ‘еҷЁиҷҪз„¶еҸҜд»Ҙе®һзҺ°иҝҷз§ҚжЈҖжҹҘпјҢдҪҶд№ҹзӣёеҪ“trickyгҖӮPostgreSQLжҸҗдҫӣзҡ„EXCLUDEзәҰжқҹеҸҜд»Ҙдјҳйӣ…ең°и§ЈеҶіиҝҷдёҖзұ»й—®йўҳгҖӮ

EcludeзәҰжқҹзҡ„иҜӯжі•

exclude_element in an EXCLUDE constraint is:
{ column_name | ( expression ) } [ opclass ] [ ASC | DESC ] [ NULLS { FIRST | LAST } ]

EXCLUDEеӯҗеҸҘе®ҡдёҖдёӘжҺ’йҷӨзәҰжқҹпјҢе®ғдҝқиҜҒеҰӮжһңд»»ж„ҸдёӨиЎҢеңЁжҢҮе®ҡеҲ—жҲ–иЎЁиҫҫејҸдёҠдҪҝз”ЁжҢҮе®ҡж“ҚдҪңз¬ҰиҝӣиЎҢжҜ”иҫғпјҢдёҚжҳҜжүҖжңүзҡ„жҜ”иҫғйғҪе°Ҷдјҡиҝ”еӣһTRUEгҖӮеҰӮжһңжүҖжңүжҢҮе®ҡзҡ„ж“ҚдҪңз¬ҰйғҪжөӢиҜ•зӣёзӯүпјҢиҝҷе°ұзӯүд»·дәҺдёҖдёӘUNIQUEзәҰжқҹпјҢе°Ҫз®ЎдёҖдёӘжҷ®йҖҡзҡ„е”ҜдёҖзәҰжқҹе°Ҷжӣҙеҝ«гҖӮдёҚиҝҮпјҢжҺ’йҷӨзәҰжқҹиғҪеӨҹжҢҮе®ҡжҜ”з®ҖеҚ•зӣёзӯүжӣҙйҖҡз”Ёзҡ„зәҰжқҹгҖӮдҫӢеҰӮпјҢдҪ еҸҜд»ҘдҪҝз”Ё&&ж“ҚдҪңз¬ҰжҢҮе®ҡдёҖдёӘзәҰжқҹпјҢиҰҒжұӮиЎЁдёӯжІЎжңүдёӨиЎҢеҢ…еҗ«зӣёдә’иҰҶзӣ–зҡ„еңҶпјҲи§Ғ Section 8.8пјүгҖӮ

    user_id INTEGER,
    room_id INTEGER,
    range   tsrange,
    EXCLUDE USING GIST(room_id WITH = , range WITH &&)
);

иҝҷйҮҢEXCLUDE USING GIST(room_id WITH = , range WITH &&)жҢҮжҳҺдәҶдёҖдёӘжҺ’е®ғзәҰжқҹпјҡдёҚе…Ғи®ёеӯҳеңЁroom_idзӣёзӯүпјҢдё”rangeзӣёдә’йҮҚеҸ зҡ„еӨҡжқЎи®°еҪ•гҖӮ

-- з”ЁжҲ·1йў„е®ҡдәҶ101еҸ·жҲҝй—ҙпјҢд»Һж—©дёҠ10зӮ№еҲ°дёӢеҚҲ6зӮ№

-- з”ЁжҲ·2д№ҹе°қиҜ•йў„е®ҡ101еҸ·жҲҝй—ҙпјҢдёӢеҚҲ4зӮ№еҲ°дёӢеҚҲ6зӮ№
INSERT INTO meeting_room(user_id, room_id, range) 
VALUES (2,101, tsrange('2019-01-01 16:00', '2019-01-01 18:00'));

-- з”ЁжҲ·2зҡ„йў„е®ҡжҠҘй”ҷпјҢиҝқиғҢдәҶжҺ’е®ғзәҰжқҹ
ERROR:  conflicting key value violates exclusion constraint "meeting_room_room_id_range_excl"

иҝҷйҮҢзҡ„EXCLUDEзәҰжқҹдјҡиҮӘеҠЁеҲӣе»әдёҖдёӘзӣёеә”зҡ„GISTзҙўеј•пјҡ

"meeting_room_room_id_range_excl" EXCLUDE USING gist (room_id WITH =, range WITH &&)

еә”з”ЁжЎҲдҫӢпјҡзЎ®дҝқIPзҪ‘ж®өдёҚйҮҚеӨҚ

жңүдёҖдәӣзәҰжқҹжҳҜзӣёеҪ“еӨҚжқӮзҡ„пјҢдҫӢеҰӮзЎ®дҝқиЎЁдёӯзҡ„IPиҢғеӣҙдёҚеҸ‘з”ҹйҮҚеҸ пјҢзұ»дјјзҡ„пјҢзЎ®дҝқең°зҗҶеҢәеҲ’иЎЁдёӯеҗ„дёӘеҹҺеёӮзҡ„иҫ№з•ҢдёҚдјҡйҮҚеҸ гҖӮдј з»ҹдёҠиҰҒе®һзҺ°иҝҷз§ҚдҝқиҜҒжҳҜзӣёеҪ“еӣ°йҡҫзҡ„пјҡиӯ¬еҰӮUNIQUEзәҰжқҹе°ұж— жі•иЎЁиҫҫиҝҷз§ҚиҜӯд№үпјҢCHECKдёҺеӯҳеӮЁиҝҮзЁӢжҲ–иҖ…и§ҰеҸ‘еҷЁиҷҪз„¶еҸҜд»Ҙе®һзҺ°иҝҷз§ҚжЈҖжҹҘпјҢдҪҶд№ҹзӣёеҪ“trickyгҖӮPostgreSQLжҸҗдҫӣзҡ„EXCLUDEзәҰжқҹеҸҜд»Ҙдјҳйӣ…ең°и§ЈеҶіиҝҷдёӘй—®йўҳгҖӮдҝ®ж”№жҲ‘д»¬зҡ„geoipsиЎЁпјҡ

create table geoips
(
  ips          inetrange,
  geo          geometry(Point),
  country_code text,
  region_code  text,
  city_name    text,
  ad_code      text,
  postal_code  text,
  EXCLUDE USING gist (ips WITH &&) DEFERRABLE INITIALLY DEFERRED 
);

вҖӢ иҝҷйҮҢEXCLUDE USING gist (ips WITH &&) зҡ„ж„ҸжҖқе°ұжҳҜipsеӯ—ж®өдёҠдёҚе…Ғи®ёеҮәзҺ°иҢғеӣҙйҮҚеҸ пјҢеҚіж–°жҸ’е…Ҙзҡ„еӯ—ж®өдёҚиғҪдёҺд»»дҪ•зҺ°еӯҳиҢғеӣҙйҮҚеҸ пјҲ&&дёәзңҹпјүгҖӮиҖҢDEFERRABLE INITIALLY IMMEDIATE иЎЁзӨәеңЁиҜӯеҸҘз»“жқҹж—¶еҶҚжЈҖжҹҘжүҖжңүиЎҢдёҠзҡ„зәҰжқҹгҖӮеҲӣе»әиҜҘзәҰжқҹдјҡиҮӘеҠЁеңЁipsеӯ—ж®өдёҠеҲӣе»әGISTзҙўеј•пјҢеӣ жӯӨж— йңҖжүӢе·ҘеҲӣе»әдәҶгҖӮ
